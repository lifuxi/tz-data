# tz-data 行情数据需求

> **原则**：tz-data 只负责数据采集、存储、质量监控。所有面向用户的展示、分析、交互需求归属 tz2.0。

---

## 一、数据类型与存储

| 数据类型 | 存储表 | 数据源 | 同步周期 | 备注 |
|----------|--------|--------|----------|------|
| 日线行情 | `daily_quotes` | Tushare | 每日盘后(18:00) | 已有 |
| 分钟行情 | `minute_quotes` | Tushare | 交易日每5分钟 | 已有 |
| 期权日线 | `daily_quotes` (option_type) | Tushare | 每日盘后 | 需完善同步 |
| 期权分钟线 | `minute_quotes` (option_type) | Tushare | 交易日每5分钟 | 需完善同步 |
| 持仓量 | `daily_quotes.volume` / `position_detail` | Tushare | 每日盘后 | 已有 |
| IV/希腊字母快照 | 待建表 `option_greeks_snapshot` | Tushare | 每日盘后 | 需新建 |

## 二、数据完整性要求

### 2.1 分钟K线完整性（MKT-06 依赖）

- **要求**：近1年交易日分钟数据完整覆盖
- **覆盖品种**：IF/IH/IC/IM（股指期货）、IO/MO/HO（股指期权）、商品期货主力
- **覆盖时段**：每个交易日 09:30-11:30, 13:00-15:00（股指期货）；09:30-11:30, 13:00-15:15（股指期权）
- **缺失检测**：每日 18:50 的 `detect_data_gaps` Celery 任务需覆盖分钟级数据

**涉及文件**：
- `src/tzdata_pkg/scheduler/tasks/check_tasks.py` — `detect_data_gaps` 任务
- `src/tzdata_pkg/maintenance/metadata/trade_calendar.py` — 交易日历

### 2.2 期权数据同步

- **需求**：Tushare 期权日线/分钟线数据同步
- **覆盖**：IO（沪深300期权）、MO（中证1000期权）、HO（上证50期权）
- **字段**：合约代码、交易日、开高低收、成交量、持仓量、结算价、行权价、到期日

**涉及文件**：
- `src/tzdata_pkg/maintenance/sources/tushare_source.py` — Tushare 数据源
- `src/tzdata_pkg/scheduler/tasks/mo_tasks.py` — MO 相关同步任务

### 2.3 K线数据质量校验

- **需求**：K线数据连续性检查、异常值检测
- **检查项**：
  - 交易日有数据 vs 无数据（对比交易日历）
  - 价格字段非零检查
  - 极值检测（日涨跌 > 9%）
  - 成交量异常检测
  - 与 settle 价格差异检测

**涉及文件**：
- `src/tzdata_pkg/maintenance/monitoring/completeness_checker.py` — 完整性检查
- `src/tzdata_pkg/maintenance/monitoring/quality_evaluator.py` — 质量评估

### 2.4 IV/希腊字母快照

- **需求**：期权隐含波动率和希腊字母（Delta/Gamma/Vega/Theta）日级快照
- **存储**：新建 `option_greeks_snapshot` 表或复用已有结构
- **字段**：合约代码、交易日、IV、Delta、Gamma、Vega、Theta、Rho

**涉及文件**：
- `src/tzdata_pkg/storage/db_registry.py` — 数据库表定义
- 新建同步任务获取希腊字母数据

## 三、与 tz2.0 的数据接口

tz-data 通过以下方式向 tz2.0 提供数据：

| 接口 | 说明 |
|------|------|
| SQLite 直读 | tz2.0 直接读取 `daily_quotes` / `minute_quotes` 表 |
| tzdata_pkg API | tz2.0 通过 Python import 调用 `tzdata_pkg` 内部函数 |

tz-data 不直接提供面向用户的 RESTful API。所有面向用户的 API 端点在 tz2.0 的 `src/api/market_data.py` 中实现。
